# zig_sdl3_sample01

# SDL:
 - Version 3.1
 
# Zig:
 - Version 0.12.0

# Information:
 Note that SDL2 to SDL3 has change API.

 Since some area almost the same but the functions and params have change.

# Set Up:
 Download SDL3 from github.
```
-lib // zig default
--sdl3 // folder
---include
---lib
```
It config in build.zig
